Here are some tags to note in HTML5

The <strong> tag represents importance semantically in html5
The <u> tag represents "unarticulated", a run of text that you want to call attention to because it is wrong. Should not be used for merely underline purposes.
The <mark> tag is used for highlighting a portion of text for reference purposes. An example would be highlight of words that match a search criteria.
The <em> tag is used to stress emphasis on a run of text. Or used for keywords.
The <small> tag is useful for small prints such as disclaimer, legal restrictions or a copyright.

The <blockquote> basically is used to imply a quote.
The <s> tag is used for contents that are no longer valid but left in place for reference. An example below:

Buy our Iced Tea and Lemonade

Recommended retail price: $3.99 per bottle

Now selling for just $2.99 a bottle


The <pre> element ensures the indentation stays in place else html will not pay attention to any whitespace.
